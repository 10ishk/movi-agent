# Future Scope

The current implementation is a local full-stack prototype. The following items are planned enhancements rather than completed features.

## AI Agent Roadmap

- LangGraph-based workflow orchestration
- Stronger multi-step dialog state
- Persistent agent memory for pending actions
- Better tool routing and structured action traces

## Multimodal Roadmap

- OCR-based screenshot parsing
- OpenAI Vision or similar multimodal model support
- More reliable screenshot-to-trip matching
- OpenAI Realtime API or equivalent low-latency voice interaction

## Product Roadmap

- Authentication and role-based access control
- Persistent audit logs
- Database-backed confirmation workflow
- Admin analytics for AI usage and failed queries
- Docker Compose setup
- Deployment documentation
- Automated tests

## Data and Security Roadmap

- Stronger input validation
- PII-safe logging
- Environment-based configuration
- Production secret management
- Access-controlled operational actions
